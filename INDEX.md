# BaarnsMannenKoor

## What it is
A modern static website for the Baarns MannenKoor, a men's choir in Baarn founded in 1907. Replaces the old baarnsmannenkoor.nl with an Astro site hosted on GitHub Pages. Future plans include ticket sales and downloadable sheet music.
